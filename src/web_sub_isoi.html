반응형 웹 리뉴얼

착한 성분만을 사용하는 아이소이의 철학을 담아 홈페이지를 리뉴얼 했습니다.
최대한 깔끔하게 디자인 하여 페이지의 가독성을 높이고,
기존 웹/모바일로 나누어져 있던 페이지를 반응형으로 제작함으로써
어느 기기에서든 편하게 볼 수 있도록 리뉴얼 하였습니다.

홈페이지 바로가기

다양한 디바이스에 효과적으로 대응하는 반응형 웹
해당 페이지는 언제 어디서든 다양한 디바이스에 효과적으로 대응하는 반응형 웹페이지로 제작 되었습니다.
